# Codex Projects

This repository contains small projects built with Codex. The current project is **Viral Clicker**, an idle clicker game implemented with vanilla HTML, CSS and JavaScript.

## Running the Game

Open `index.html` in a web browser. Alternatively, run a local HTTP server from this directory:

```bash
python3 -m http.server
```

Then navigate to `http://localhost:8000` in your browser.

The game saves progress automatically in `localStorage` every few seconds.

## Features

- Ten upgrade tiers that generate coins automatically
- Colorful animations for clicks and a shifting background
- Achievements that unlock as you play
- Prestige system: reset after 1,000,000 coins for a permanent 10% CPS boost
